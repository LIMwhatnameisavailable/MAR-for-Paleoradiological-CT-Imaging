# MAR_for_Paleo_CT/data 文件结构说明

## 总体概况

数据位于 `MAR_for_Paleo_CT/data/常州古尸CT/`，共 **8 个标本**，约 **14.5 GB，16,000+ 文件**，均为 Philips Ingenuity CT 扫描输出的 DICOM 格式文件（无扩展名，含 DIRFILE 索引）。

标本编号格式 S917xx / S918xx，PatientID 统一为 CZGS-xxx（常州古尸缩写）。扫描参数：120kV/100kV，iDose 迭代重建（3~6级），RescaleIntercept = -1024，RescaleSlope = 1（像素值可直接转为 HU）。

### 目录层级

```
常州古尸CT/S917xx/            ← 标本 ID
             ├── S1xxx/       ← 定位片 (Localizer)
             ├── S2xxx/       ← 轴向扫描体数据
             ├── S3xxx/       ← 轴向扫描体数据
             ├── S4xxx~S8xxx  ← 其他序列或 Exam Summary
             └── 各序列下：
                   ├── DIRFILE  ← 目录索引文件
                   ├── I10      ← 切片数据（步长 10 编号）
                   ├── I20
                   └── ...
```

## MAR 相关总结

- **含 O-MAR 配对数据的标本（可作真值参照）**：S91800、S91810
- **金属伪影严重但无 O-MAR 的标本（适合无参照算法开发）**：S91760
- **有薄层体数据、适合 MAR 处理的标本**：S91740（1mm×405）、S91800（1mm×1130）、S91810（1mm×750）
- **仅有厚层数据、参考价值一般的标本**：S91730、S91770、S91780
- **不含轴向体数据的标本**：S91790

# 方案 A：有原始 Sinogram 数据

**核心思路**：在投影域修正测量误差本身，从根源消除射线硬化，信息损失最小。

---

```
[原始 DICOM + Sinogram Raw Data]
        │
        ▼
[阶段一] 汞区域分割
  · 阈值分割：Hg ~3000-4000 HU，HgS ~1500-2500 HU（额外考虑混合态密度偏差）
  · 形态学后处理（连通域分析 + 边界膨胀 3-5px）
  · 手动逐层审核；分别生成 D_Hg 掩膜 / D_HgS 掩膜
  ※ Kim et al. §6.2：分割精度是所有线性化方法的核心敏感点，
    Hg 与 HgS 需分别建掩膜，因两者 ρ_j、Z_j、A_j 不同，
    将分别代入 §3.2 的 α_j（光电项）和 β_j（Compton项）参数计算。
        │
        ▼
[阶段二] 投影域物理射线硬化校正（Kim et al., 2026）

  Step 1｜初始重建
    · 对原始 sinogram p(θ̂,s) 直接 FBP 重建，得含伪影初始图像。
    · 依据 Kim et al. §2.1，Beer-Lambert 测量模型：
        p(θ̂,s) = -ln N(θ̂,s)
        N(θ̂,s) = ∫ φ̄(E) exp[-∫ μ(r,E) dℓ] dE
      FBP 假设单色 X 射线，多色谱导致 cupping 与条纹伪影。

  Step 2｜路径长度计算
    · 对每条射线路径，计算穿过各材料区域的路径长度：
        |L(θ̂,s) ∩ D_j|（汞/HgS/骨骼）
        |L(θ̂,s) \ D|（水等效软组织区 D₀）
    · 代入各材料物理参数（Kim et al. §3.2）：
        Hg：Z=80，ρ=13.6 g/cm³，A=200.59
        HgS：Z_eff≈74，ρ≈8.1 g/cm³
        软组织：水等效，Z≈7.4，ρ≈1.0 g/cm³

  Step 3｜求解路径依赖平均能量 Ẽ(θ̂,s)
    · Kim et al. §2.2，由积分中值定理存在 Ẽ 使得：
        p(θ̂,s) = ∫ μ(r,Ẽ) dℓ - ln[φ̄(Ẽ)|I|]
    · §2.3，诊断能量段（30-140 keV）中衰减系数近似为：
        μ(r,E) ≈ a_p(r)·f_p(E) + a_c(r)·f_KN(τ)
      其中 f_p(E) ≈ 1/E^ξ（光电项），f_KN 为 Klein-Nishina 公式。
    · §3.1，对 f_KN 作幂级数展开后，Ẽ 满足 (k+ξ) 次多项式方程：
        Σ α_j|L∩D_j| - [p_m + φ̄'(E_n)|I|(Ẽ-E_n)
          - (Σ β_j|L∩D_j|)·f_KN^(k)] · Ẽ^ξ = 0
      逐路径数值求解，得 Ẽ(θ̂,s)。

  Step 4｜修正 sinogram 并重建
    · Kim et al. §4 Step 7，校正公式：
        p_cor = p + ln[φ̄(Ẽ)|I|]
              + (Ẽ^ξ - E_n^ξ)/(E_n^ξ · Ẽ^ξ)
                · [a_p^w|L\D| + Σ 各材料项]
    · 对 p_cor 执行 FBP 重建，得第一次校正图像。

  Step 5｜Re-correction（迭代一次）
    · 用第一次校正图像细化分割（软组织对比度提升后边界更准）
    · 重复 Step 2-4，得最终校正图像。
    · Kim et al. Table 1 显示 Re-correction 使 ROI 3 的 MAE
      从 386 降至 190，SNR 从 0.70 提升至 1.81，收益显著。
        │
        ▼
[阶段三-A] 高能量 monoE 融合
  条件可选：需双能 CT 数据。飞利浦 Ingenuity 为单能扫描，
     本数据集大概率不可用，跳过此步骤。
  · 若可用：推荐 110-140 keV 虚拟单能图像融合，压制残余条纹。
        │
        ▼
[阶段三-B] 深度学习伪影纹理抑制
   木乃伊器官严重萎缩、脱水、形态与现代活体有根本性差异，
     用现代 CT 先验训练的模型会将真实古代病理特征误判为伪影并"修正"。
  · 正确定位：仅学习条纹伪影的纹理模式，执行伪影纹理抑制。
  · 训练数据方向（物理校正后）：含金属伪影的 CT 配对（学伪影纹理而非解剖内容），
    或同批次其他古尸 CT（S91740 等）作为域内先验。
  · 汞密集区域内部：信息已在重建时损坏，不执行 inpainting，
    直接标注为"不可解释区域"——这是学术上更诚实的处理方式。
        │
        ▼
[阶段四] 诊断目标区域专项分析
  · 主动脉窦 ROI 提取
  · 斑块 CNR 定量评估（2mm×3mm 和 3mm×5mm 目标斑块）
  · 软组织窗 / 血管窗多窗宽观察
        │
        ▼
[阶段五] 结果验证
  · 骨骼 HU 回归验证（皮质骨 700-1200 HU，松质骨 150-400 HU，
    偏差 < 10% 认为校正方向正确）
  · 投影域一致性验证：p_cor 与 p 的残差分布应无系统性偏移
  · 解剖合理性盲审（不以"正常"为标准，以古尸自身解剖逻辑为准）
  · 与原始图像逐层对比溯源；S91800/S91810 O-MAR 配对数据作参照
```

---

| 维度 | 评估 |
|------|------|
| 汞区域内部恢复能力 | ⚠️ 投影域可部分恢复，但密集区域仍标注为不可解释 |
| 汞区域外部恢复质量 | ✅ 高 |
| 理论上限 | ✅ 高 |

## 方案 B：仅有重建后 DICOM 图像

**核心思路**：在图像域近似校正误差表现，汞密集区域承认信息损失并用 inpainting 替代，优先保障汞区域外的软组织质量。

```
[重建后 DICOM 图像]
        │
        ▼
[阶段零] 前置确认（优先执行）
  · 在 3D Slicer 中定位主动脉窦位置
  · 测量目标区域的伪影程度和 HU 偏移
  · 判断斑块是否落在汞伪影"死区"内
  · → 决定后续需要投入的技术力度
        │
        ▼
[阶段一] 汞区域分割（与方案 A 相同）
  · 阈值分割 + 形态学后处理 + 手动审核
  · 额外生成：伪影辐射影响区掩膜（汞边界外扩 20-30px）
        │
        ▼
[阶段二] 图像域射线硬化校正（三选一或组合）

  · 选项 1 — EBHC 变体（优先推荐）
      估算每像素的等效汞路径长度
      用骨骼 HU 偏移量反推多项式校正系数
      将校正函数应用至软组织区域

  · 选项 2 — 飞利浦 O-MAR（快速 baseline）
      数据本身为飞利浦 Ingenuity 扫描，直接兼容
      参考 S91800 O-MAR 配对结果预判局限性

  · 选项 3 — iMAR / SEMAR（对比参照）
      需要对应厂商工作站访问权限
      用于与选项 1/2 结果横向比较
        │
        ▼
[阶段三] 深度学习伪影纹理抑制

  · 汞区域外：仅做条纹伪影纹理抑制
      训练数据：含金属伪影CT配对 或 同批古尸CT（S91740等）
      不使用现代正常腹腔CT先验

  · 汞密集区域内：直接标注为"不可解释区域"
      无可靠先验，不执行 inpainting，不纳入定量诊断

        │
        ▼
[阶段四] 诊断目标区域专项分析（与方案 A 相同）
  · 主动脉窦 ROI 提取
  · 斑块 CNR 定量评估
  · 软组织窗 / 血管窗多窗宽观察
        │
        ▼
[阶段五] 结果验证
  · 骨骼 HU 回归验证（核心间接 ground truth）
      皮质骨应收敛至 700-1200 HU
      松质骨应收敛至 150-400 HU
      偏差 < 10% 认为校正方向正确
  · 解剖合理性盲审
  · 与原始图像逐层对比溯源
  · S91800 / S91810 O-MAR 配对数据作评估参照
  · Inpainting 区域单独标注，不纳入定量诊断
```

| 维度 | 评估 |
|------|------|
| 汞区域内部恢复能力 | ❌ 基本无法恢复，用 inpainting 替代 |
| 汞区域外部恢复质量 | ⚠️ 中等 |
| 理论上限 | ⚠️ 中 |
